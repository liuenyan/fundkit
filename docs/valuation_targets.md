# 估值定投标的候选列表

> 筛选条件：中证指数 + 指数成立 ≥10 年 + 跟踪基金成立 ≥5 年 + 规模 ≥5 亿
> 数据截止: 2026-06-29

## 数据源覆盖说明

估值数据源的覆盖度是选择标的的核心约束：

| 层级 | 数据源 | 覆盖范围 | 精度 |
|------|--------|---------|------|
| **Level 1** | `stock_index_pe_lg` | **仅沪深300/中证500/中证800**（20+年日频PE） | PE百分位最精确 |
| **Level 2** | `stock_zh_index_daily_em` | 数千只指数日线（含中证/国证/标普/恒生） | 价格MA偏离度 |
| **Level 3** | 基金累计净值 | 所有基金 | acc_nav MA偏离度 |

中证行业指数（消费/医药/银行/白酒等）仅能获取最近20个交易日的PE快照，无法计算有统计意义的百分位。因此**PE百分位策略只适用于沪深300、中证500、中证800**三只宽基指数。

## 关于中证行业指数PE数据的说明

中证行业指数（消费/医药/银行/酒/军工等）在AKShare上**没有长期PE历史**。`stock_zh_index_value_csindex` 仅返回最近20个交易日快照，无法计算有意义的百分位。

替代方案：

```
PE百分位不可用 ──→ 指数价格MA偏离度（Level 2）
                    ↓
               `stock_zh_index_daily_em` 有全部指数日线
                    价格MA不同，但均值回归逻辑同样成立
```

---

## 使用建议

1. **定投核心仓位**：沪深300 + 中证500，PE百分位与价格MA双信号可用
2. **行业轮动仓位**：中证消费/医药/银行/白酒等，使用价格MA偏离度
3. **高分红标的**：中证红利/红利低波，注意改用 acc_nav MA（见 nav_types_analysis.md）
4. **低流动性标的**：基建/钢铁/通信等规模较小的指数，建议暂不纳入
